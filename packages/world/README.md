# @worlddesc/world

TypeScript package for loading, validating and running `worlddesc` worlds.

It contains:

- world loaders and validation
- runtime creation
- player-facing view models
- asset loading
- narrative guide and guide-mix loading

CLI entry points:

- `npx @worlddesc/world checkworld ./sample/test.world.yaml`
- `npx @worlddesc/world checkasset ./sample/assets/safe.object-asset.yaml`
